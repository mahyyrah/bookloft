BOOKLOFT

Project Description
BOOKLOFT is a simple web-based book management system developed using HTML, CSS, and JavaScript. The system allows users to register, log in, manage book information, and navigate through various pages such as Dashboard, About Us, and Contact.

Features Included
- User Registration
- User Login
- Dashboard Page
- Add Book Functionality
- About Us Page
- Contact Page
- Logout Functionality
- Responsive User Interface

Instructions to Test Login
1. Open the live website.
2. Register a new account through the Register page.
3. Log in using the registered username and password.
4. After successful login, the user will be redirected to the Dashboard page.
5. Users can add book information through the Add Book page.

Frameworks/Libraries Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
